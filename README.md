# Booking flights and rooms app

In this project i tried to make an application where you can book flights and room in different cities and countries.
My inspiration for this application is Booking.

Application is made in *C#*,exactly in Windows Forms.

#Features#

*You can book one way flights and return flights
*Account system,where you can make a simple account to use applications services
*Flights,rooms and taxies features need an account to use
*Warnings menu,for different hostiles situations in different destinations
*Atractions menu,where you cand see different atractions in some cities
*Trip calculator,where you can make a buget for your trip
